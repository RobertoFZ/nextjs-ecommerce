This is a [Next.js](https://nextjs.org/) boilerplate project

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the page.

## Technology

- MaterialUI
- Husky
- Prettier
- ESLint
- Styled components
